Web Apps Collection

日常や仕事で役立つ、シンプルで便利な10種類のWebツールを開発しました。すべて無料で、ブラウザからインストール不要ですぐに利用できます。

🤖

01: Auto Prompt Builder

AI（ChatGPTやClaudeなど）への指示を最適化するプロンプト作成ツール。目的や条件を入力するだけで、質の高い回答を引き出すプロンプトを自動生成します。

📝

02: Text Analyzer PRO

テキストを貼り付けるだけで、文字数・読了時間・漢字やひらがなの構成比を瞬時に解析。ライターやブロガーのための高度な文章分析ツールです。

🎨

03: Color Palette PRO

ベースカラーを1つ選ぶだけで、補色・類似色・トライアドなど5種類の美しい配色パターンを自動生成します。デザイナー向けの色彩支援ツールです。

📱

04: QR Code Generator

URLやテキストから一瞬でQRコードを生成。色やサイズのカスタマイズ、ロゴの埋め込みにも対応し、高画質でダウンロード可能です。

🔐

05: Password Generator

文字数や使用する文字種（英数字・記号など）を指定して、安全性の高い強固なパスワードを瞬時に生成。セキュリティ強度も一目で確認できます。

⚖️

06: Unit Converter

長さ、重さ、面積、体積など、様々な単位を相互に変換できるツール。直感的なインターフェースで、世界中のあらゆる単位計算を簡単に。

🍅

07: Pomodoro Timer

25分の集中と5分の休憩を繰り返す「ポモドーロ・テクニック」専用タイマー。タスク管理機能も備え、あなたの生産性を劇的に向上させます。

🎂

08: Age Calculator

生年月日を入力するだけで、正確な年齢、生まれてからの総日数、次の誕生日までの日数を瞬時に計算。履歴書の作成や記念日の確認に便利です。

✍️

09: Markdown Editor

ブラウザ上で動作する軽量なMarkdownエディタ。リアルタイムプレビュー機能付きで、書いた内容をすぐにHTMLやPDFとして出力可能です。

✨

10: CSS Gradient Generator

直感的な操作で美しいCSSグラデーションを作成できるツール。複数の色や角度を調整し、ワンクリックでCSSコードをコピーできます。

👶

11: Baby Log

2025年12月生まれの赤ちゃんのための育児管理。授乳・オムツ・睡眠を片手で爆速記録。夫婦間共有のための設定も備えたモダンな育児ログ。
